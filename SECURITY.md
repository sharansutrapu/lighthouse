# Security Policy

## Supported Versions

The following versions of LightHouse are currently supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of LightHouse very seriously. If you discover a security vulnerability, **please do NOT report it by opening a public issue on GitHub.**

Instead, we ask that you privately report it to us so we can verify and patch the issue before it is publicly disclosed.

**Please send an email to: security@lighthouses.digital** (or directly contact the maintainers at the email provided on our official website). 

We will triage your report and respond to you as quickly as possible, usually within 48 hours.

### What to include in your report:
- A detailed description of the vulnerability.
- Steps to reproduce the issue.
- Any potential impact on the system or users.
- Your contact information for follow-up.

Thank you for helping keep LightHouse secure and for practicing responsible disclosure!
